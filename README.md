# Serkan Gurel - Portfolio Website

Personal portfolio website for Serkan Gurel, Mechanical Engineer specializing in Defense Industry.

## Quick Start

### Option 1: VS Code Live Server (Recommended)

1. Open this folder in VS Code
2. Install the "Live Server" extension (will be recommended automatically)
3. Right-click on `index.html` and select "Open with Live Server"
4. Website will open at `http://localhost:5500`

### Option 2: Using npm

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Website will open at `http://localhost:3000`

### Option 3: Direct Browser

Simply double-click `index.html` to open directly in your browser.
